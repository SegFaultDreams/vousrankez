IA & Prompting

Prompt — Définition simple

L'instruction que vous donnez à une IA : la clé pour obtenir les bons résultats.

Un prompt est une instruction, question ou consigne soumise à une intelligence artificielle générative (ChatGPT, Claude, Gemini...) pour obtenir une réponse, un texte, une image ou un code. La qualité du prompt détermine directement la qualité du résultat : "garbage in, garbage out".

Les composantes d'un bon prompt

  • Le rôle : donnez un contexte à l'IA. "Tu es un expert en communication pour les PME industrielles..."
  • La tâche : ce que vous voulez obtenir. "Rédige un post LinkedIn..."
  • Le contexte : les informations nécessaires. "Mon entreprise fabrique des pièces métalliques à Valenciennes, ma cible est les directeurs d'usine..."
  • Le format : comment vous voulez le résultat. "En 3 paragraphes, avec un titre accrocheur, 300 mots maximum..."
  • Le ton : "Ton professionnel mais accessible, sans jargon technique..."

Exemples de prompts efficaces

Prompt faible : "Écris un email."

Prompt efficace : "Tu es un expert en communication B2B. Rédige un email de relance pour une PME de services informatiques basée dans le Nord. Le destinataire est un directeur financier qui a téléchargé notre guide il y a 2 semaines sans donner suite. Ton : professionnel et direct, sans pression. 150 mots maximum. Propose 3 variantes d'objet."

Le prompt engineering

Le prompt engineering est la discipline qui consiste à maîtriser l'art de rédiger des prompts efficaces pour obtenir les meilleurs résultats des IA. Ce n'est pas de la programmation, c'est de la communication — expliquer clairement ce qu'on veut, avec le bon contexte, à une machine très puissante mais littérale.

C'est une compétence transférable : mieux vous savez structurer vos demandes à une IA, plus elle devient utile pour votre travail quotidien.
