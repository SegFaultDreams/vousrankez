Meta Description — Définition simple

Le texte qui s'affiche sous votre lien dans Google. 160 caractères pour convaincre de cliquer.

La meta description est un court texte (idéalement entre 150 et 160 caractères) qui décrit le contenu d'une page web. Elle apparaît sous le titre dans les résultats de recherche Google. Elle n'influence pas directement le classement SEO, mais elle impacte fortement le taux de clic (CTR).

Où apparaît la meta description ?

Voici ce que voit l'internaute dans Google :

  • Titre cliquable (balise title) : en bleu, le titre de votre page
  • URL : l'adresse de votre page
  • Meta description : le texte de description en gris en dessous

C'est votre "accroche publicitaire" dans les résultats Google. Si elle n'est pas convaincante, les internautes cliquent sur vos concurrents même si vous êtes mieux classé.

Comment rédiger une meta description efficace

  • Maximum 160 caractères : au-delà, Google tronque le texte avec "..."
  • Inclure le mot-clé principal : Google le met en gras dans les résultats, ce qui attire l'œil
  • Décrire le bénéfice : pas juste ce que contient la page, mais ce que l'utilisateur va y trouver
  • Inclure un appel à l'action : "Découvrez", "Apprenez", "Téléchargez"...
  • Chaque page a sa propre meta description : ne copiez-collez pas la même partout

Meta description et SEO : attention à l'idée reçue

La meta description n'est pas un facteur de classement direct : avoir votre mot-clé dedans ne vous fera pas monter dans les résultats. En revanche, elle influence le taux de clic, et un bon CTR peut indirectement améliorer votre classement (Google observe que les gens cliquent plus souvent sur vous).

Si vous ne rédigez pas de meta description, Google en génère une automatiquement à partir du contenu de la page. Ce n'est pas toujours pertinent — mieux vaut la contrôler vous-même.
